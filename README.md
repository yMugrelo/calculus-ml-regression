## Descrição do Projeto

Este projeto integra conceitos de **Cálculo Diferencial** com métodos de **Aprendizado de Máquina**, demonstrando como técnicas de regressão podem ser utilizadas para aproximar funções reais e analisar seu comportamento.

Com base em dados experimentais de aceleração, o projeto tem como objetivos:

- Modelar o comportamento dos dados por meio de **regressão linear** e **regressão polinomial**;
- Calcular **derivadas numéricas** a partir dos modelos obtidos;
- Comparar o desempenho das técnicas de regressão na previsão e análise da função original;
- Demonstrar, na prática, como o cálculo diferencial pode ser aplicado em conjunto com ferramentas computacionais para interpretação de dados reais.
