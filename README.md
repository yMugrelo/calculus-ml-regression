**Disciplina:** Cálculo I

##  Descrição do Projeto

Este trabalho tem como objetivo demonstrar como técnicas de **regressão** (linear e polinomial) podem ser aplicadas para **aproximar funções reais** e **calcular suas derivadas numericamente**.  
A proposta une conceitos de **Cálculo Diferencial** e **Aprendizado de Máquina**, mostrando na prática como a modelagem pode estimar o comportamento de fenômenos físicos.

Os dados utilizados representam medições de aceleração ao longo do tempo, e foram analisados para construir um modelo que aproxima a função original.  
A partir desse modelo, derivamos numericamente a função e comparamos os resultados com a derivada analítica, avaliando erros e limitações.

---

##  Objetivos

- Aplicar **regressão linear e polinomial** em dados experimentais.  
- Calcular **derivadas numéricas e simbólicas** com Python.  
- Comparar resultados com **derivadas analíticas** obtidas no Cálculo.  
- Discutir conceitos de **overfitting, underfitting** e **erro quadrático médio (MSE)**.  

---

- **Python**
- **NumPy** – operações matemáticas e vetoriais  
- **Matplotlib** – visualização de dados  
- **Scikit-learn** – regressão linear e polinomial  
- **Sympy** – derivadas simbólicas  
- **Jupyter Notebook** – desenvolvimento e demonstração dos resultados

##  Project Description

This project aims to demonstrate how **regression techniques** (linear and polynomial) can be applied to **approximate real-world functions** and **compute their derivatives numerically**.  
It connects concepts from **Differential Calculus** and **Machine Learning**, showing how data modeling can estimate physical or mathematical phenomena.

The dataset used contains acceleration measurements over time.  
A regression model is built to approximate the original function, then derived numerically and symbolically to compare with the analytical derivative.  
The goal is to evaluate how well machine learning can replicate classical calculus results.

---

##  Objectives

- Apply **linear and polynomial regression** to real or simulated data.  
- Compute **numerical and symbolic derivatives** using Python.  
- Compare results with **analytical derivatives** from calculus.  
- Discuss **overfitting, underfitting**, and **error metrics (MSE, R²)**.
